# First Non-Repeating Character
### Find the first non-repeating character in the string of lower case letters.

&emsp; 
    <br>  <b> Input : </b>
    <br>  &emsp;  &emsp;  &emsp; abcdcaf
    <br> <br> <b> Output :  </b>
    <br>  &emsp;  &emsp;  &emsp; 1
    
<hr> Solution :  Using Hash Table -> &emsp; &emsp; &ensp; O(N) Time | O(1) Space
