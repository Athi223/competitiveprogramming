# Partitioning Linked List

## INPUT FORMAT
```
1st line - Length of Linked List
2nd line - Linked List values
3rd line - Index to be partitioned (k)
```

## OUTPUT FORMT
```
Linked List 1 : Values
Linked List 2 : Values
```

## Example
```
INPUT - 5
        1 2 3 4 5
        3
OUTPUT - Linked List 1 : 1 2 3
         Linked List 2 : 4 5
```
